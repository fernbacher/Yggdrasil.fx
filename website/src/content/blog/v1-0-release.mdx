# Yggdrasil.fx v1.0 — Initial Release

After months of development and testing across Vulkan, D3D9, and Proton, **Yggdrasil.fx v1.0** is now public.

## What's Included

The suite ships with **16 shaders** covering the full post-processing pipeline:

- **SceneKey** — Shared scene brightness pre-pass for adaptive shaders
- **Deband** — Perceptual debanding with randomised angular sampling
- **EdgeAA Temporal** — Edge-directed temporal anti-aliasing with debug modes
- **SSAO** — GTAO-inspired ambient occlusion (6 directions x 8 depth reads = 48 taps)
- **Color** — Perceptual colour grading in Oklab space with presets
- **WhiteBalance** — Kelvin temperature + green/magenta tint correction
- **LocalMean** — Shared multi-scale blur pre-pass
- **Atmos** — Atmospheric separation and dehaze
- **Clarity** — Mid-frequency local contrast enhancement
- **Sharp** — Adaptive sharpening with halo suppression
- **Tonemap** — AgX-inspired SDR display transform with temporal exposure smoothing
- **Bloom** — Multi-pass bloom (bright threshold -> gaussian blur -> composite)
- **Halation** — Filmic highlight bleed into deep shadows
- **FilmGrain** — Triangular dither with luma-weighted envelope
- **Vignette** — Radial vignette with adjustable center and softness
- **LUT** — 3D LUT applier with trilinear interpolation

## Platform Support

Tested and verified on:

- **Linux** via Proton (Vulkan)
- **Windows** (Vulkan and D3D9)
- ReShade 5.8 and later

## Load Order Matters

Shaders execute sequentially. Follow the [recommended load order](/docs) for correct pipeline behaviour — SceneKey and debanding first, detail shaders in the middle, finish effects last.

## What's Next

Future releases will focus on performance optimizations, additional debug visualizations for EdgeAA Temporal, and DX10/11/12 support.
